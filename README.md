# 💳 Credit Card Transaction Analysis Dashboard

This project analyzes **credit card transaction data** to identify revenue trends, customer spending behavior, and card usage patterns using **Power BI and DAX**. The dashboard provides insights that help understand customer segments, expenditure types, and overall financial performance.

---

# 📊 Project Objective

The main objective of this project is to:

* Analyze **credit card transaction behavior**
* Identify **revenue trends and transaction volume**
* Understand **customer spending patterns**
* Evaluate **card usage across different categories**
* Support **data-driven financial decision making**

---

# 📁 Dataset

The project uses two datasets:

### 1️⃣ Credit Card Transaction Dataset

Contains information related to:

* Revenue
* Transaction amount
* Transaction volume
* Card category
* Expenditure type
* Week number

### 2️⃣ Customer Dataset

Contains customer demographic details such as:

* Job type
* Education level
* Gender
* Customer category

Both datasets are connected using:

```
Client_Num
```

---

# 🧠 Data Modeling

A relationship was created between the two datasets to enable **customer-based transaction analysis**.

```
Customer Table
       │
       │ Client_Num
       ▼
Credit Card Transaction Table
```

This allows analysis of **customer attributes along with transaction behavior**.

# ⚙️ DAX Calculations

Several **DAX measures and calculated columns** were created to support analysis.

### Convert Week Column to Numeric

```DAX
Week_Num2 = VALUE(SUBSTITUTE([Week_Num], "Week-", ""))
```

This transformation enables **time-based analysis** of weekly revenue.

---

### Total Revenue

```DAX
Revenue = SUM(credit_card[Revenue])
```

---

### Total Transaction Amount

```DAX
Total_Transaction = SUM(credit_card[Total_Trans_Amt])
```

---

### Total Interest Earned

```DAX
Interest_Earned = SUM(credit_card[Interest_Earned])
```

---

### Week-over-Week Revenue Growth

```DAX
WoW_Revenue =
DIVIDE(
    [Current_week_Revenue] - [Previous_week_Revenue],
    [Previous_week_Revenue]
)
```
